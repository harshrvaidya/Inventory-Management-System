Inventory-Management-System

===

The Inventory Management System is built using JavaFX with a MySQL database. This desktop application is designed specifically for a pharmacy shop.



The main modules of this project include:



Login and Signup



Analytics Dashboard – includes all analytics with PDF report generation



Purchase Product Page – the shopkeeper can add the products that the customer wants, generate a temporary bill slip, and the quantity of the selected medicines is dynamically reduced from the database



Add Products Page – the shopkeeper/manager can perform full CRUD operations on medicines, including assigning shelf and compartment locations for easy retrieval



Tech Stack: JavaFX, SceneBuilder, MySQL

This project was developed in a short sprint through collaboration among 4 team members.
